# yii2-global-helpers

global shortcut functions for various yii2 components

**app()** = Yii::$app

**user()** = Yii::$app->user

**params()** = Yii::$app->params

**d()** = VarDumper::dump()

**ds()** = VarDumper::dumpAsString()

**dd()** = VarDumper::dump() and die

**e()** = echo VarDumper::export()

**es()** = return VarDumper::export()
